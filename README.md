# Repo-sample

이 저장소는 git을 연습하기 위해 만든 것

See Demo [Here](https://www.google.com/)

## Prerequisites

- Python^3.13
- git(MacOS terminal)

## How to start

```shell
$ git clone addr
$ cd Repo-sample
$ python main.py
```

## Installation

There is no installation to execute this program


## Features

```python
for _ in range(3):
    print('hello')'
```

만약 문장안에서 특정한 코드 하이라이트가 필요하다면 backquote로 감싸면 됨
